# Ett enkelt sätt att publicera Markdown utan databas

Målet med Anteckningsbiblioteket är att det ska vara enkelt att spara idéer, guider och samtal utan att behöva bygga ett stort CMS.

Den enklaste modellen är:

```text
Markdown → GitHub → Vercel → Webbplats
```

Det innebär att allt innehåll ligger kvar som vanliga Markdown-filer och versioneras automatiskt med Git.

## Varför ingen databas?

För ett personligt kunskapsbibliotek räcker Markdown väldigt långt.

Fördelar:

- Enkelt
- Snabbt
- Portabelt
- Ingen inlåsning
- All historik sparas i Git

## Nästa steg

Nästa förbättring blir ett ännu enklare publiceringsflöde där artiklar kan skapas direkt från ChatGPT och publiceras med ett klick.
